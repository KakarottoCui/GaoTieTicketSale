# GaoTieTicketSale
基于springboot+mybatis+themeleaf的高铁售票系统 

演示视频连接：https://live.csdn.net/v/204128

用到的技术

后端：Java语言，SpringBoot框架，MySQL数据库，maven依赖管理等技术；

前端：HTML页面、themeleaf框架等。

实现的功能

普通用户：注册登录、车次选择、购票下单等；

管理员：用户管理、车次管理、订单管理等。
